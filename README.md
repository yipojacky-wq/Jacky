# JASIC 股票觀測投資 App

這是一個可直接分享給朋友使用的靜態網頁 App。打開 `index.html` 即可操作，不需要安裝後端或資料庫。

## 功能

- 新增、編輯、刪除股票觀察清單
- 使用現價、目標價、成長率、本益比、動能與風險等級計算 JASIC 參考分數
- 顯示買進觀察、持續追蹤、暫緩三種訊號
- 內建圖表、搜尋、排序與統計卡片
- 使用瀏覽器 `localStorage` 保存資料
- 支援 CSV 匯入與匯出

## CSV 格式

匯入檔案第一列可使用以下欄位：

```csv
symbol,name,price,target,growth,pe,momentum,risk,notes
2330,台積電,920,1020,14.2,22,7.5,2,AI需求穩定
```

`risk` 建議填入 `1`、`2`、`3`，分別代表低、中、高風險。

## 注意

此工具只用於整理觀察資料與朋友間討論，不構成投資建議，也不保證預測結果。正式投資前仍需自行查證財報、產業資訊與市場風險。
